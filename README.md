# sparkhacks2026project
Project for SparkHacks 2026.


This project uses Vite for fast development and hot module reloading.
Prerequisites
Make sure you have the following installed:
Node.js (v16 or later recommended)
npm (comes with Node.js)

You can check with:
node -v
npm -v
in you trinal 

Installation & Setup
Clone the repository

git clone https://github.com/jera1470/sparkhacks2026project.git

Navigate into the frontend directory


cd sparkhacks2026project/frontend

Install dependencies

npm install

Start the development server


npm run dev
Open the app
Vite will output a local URL
